# vibe-check - Viral Developer Personality Quiz

## Overview

Viral zero-friction personality quiz for devs/tech workers. Traffic funnel for [VibeID](https://vibeid.dev). Inspired by SBTI: absurd self-deprecating labels, screenshot-optimized cards, zero registration.

## Tech Stack

Next.js 16 (App Router) + TypeScript + Tailwind CSS v4 + @vercel/og

## Commands

```bash
npm run dev    # Dev server
npm run build  # Production build
```

## Design Principles

- Zero registration, zero login, ~1 minute
- Self-deprecating labels only, no flattery
- Result card designed for mobile screenshot
- Funnel: fun quiz → vibeid.dev for full report
